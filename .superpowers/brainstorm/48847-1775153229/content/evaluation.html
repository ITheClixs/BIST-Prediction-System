Backtest Rules (No Cheating)

  • Walk-forward only — train on past, test on future. Window slides forward. Never peek.
  • Realistic costs — include estimated commission (0.1%) and slippage (0.05%) per trade
  • No survivorship bias — include delisted stocks that were in BIST-100 during test period
  • Signal delay — prediction made at market close, trade executed at next-day open price
  • Confidence threshold — only count trades above configurable confidence (default 60%)

Live Accuracy Tracking

  • Every prediction logged with timestamp, confidence, predicted direction and % move
  • Next-day actual result recorded automatically on next fetch
  • Rolling 30/60/90-day accuracy windows
  • Per-stock accuracy breakdown — identify which stocks the model is best/worst at
  • Confidence bucket analysis — is 80%+ confidence actually more accurate than 60-70%?
  • Automatic alerts if accuracy drops below threshold (model degradation detection)
