1. Factor Models & Alpha Signals

Fama-French (adapted) Compute SMB (size), HML (value), momentum, profitability factors for BIST-100. Rank stocks on factor exposures. Stocks with positive multi-factor alignment get alpha boost.
Cross-sectional Momentum Jegadeesh & Titman (1993). Rank all BIST-100 stocks by trailing 3/6/12-month returns. Top decile historically outperforms. Compute percentile rank as feature.
Time-series Momentum Moskowitz et al. (2012). Per-stock: if trailing 12-month excess return > 0, signal is long. Captures trend persistence. Binary signal + magnitude as features.
Mean Reversion (O-U) Fit Ornstein-Uhlenbeck process to estimate mean-reversion speed (θ), long-term mean (μ), and current deviation. Stocks far from mean with high θ → mean reversion signal.

2. Statistical Methods

Kalman Filter Track hidden "true momentum" state, filtering out noise. Outputs: filtered trend estimate, prediction error variance. Better than raw moving averages — adapts to changing volatility.
HMM Regimes 3-state Hidden Markov Model (bull / bear / sideways) trained on BIST-100 index returns + volatility. Outputs: current regime probability, regime transition probability. Controls ensemble routing.
GARCH(1,1) Per-stock volatility forecast. Conditional variance: σ²(t) = ω + α·ε²(t-1) + β·σ²(t-1). Outputs: forecasted volatility, volatility surprise (actual vs predicted). Critical for confidence calibration.
Cointegration Engle-Granger test across BIST-100 pairs. Find cointegrated pairs → when spread deviates beyond 2σ, mean reversion trade. Outputs: spread z-score, half-life of mean reversion.

3. Risk & Position Sizing

Kelly Criterion f* = (p·b - q) / b, where p = win probability, b = win/loss ratio, q = 1-p. Outputs optimal fraction of capital per trade. Use fractional Kelly (0.25-0.5×) for safety. Shown in signal output.
Ledoit-Wolf Cov. Shrinkage estimator for robust covariance matrix of BIST-100 returns. Prevents overfitting to noisy sample correlations. Used for portfolio-level risk estimation.
PCA Factors Extract top 5-10 principal components from BIST-100 return matrix. These are latent market drivers (e.g., "banking factor", "export factor"). Both features for ML and risk decomposition.

4. Signal Quality Measurement

Information Coeff. IC = rank_correlation(predicted_return, actual_return). IC > 0.05 is meaningful. Measured per feature and per model — prunes useless signals automatically.
Hurst Exponent H > 0.5 → trending (trust momentum models). H < 0.5 → mean-reverting (trust O-U / pairs). H ≈ 0.5 → random walk (reduce confidence). Computed per-stock via R/S analysis.
Wavelet Decomp. Discrete wavelet transform separates price into frequency bands: daily noise, weekly cycles, monthly trends. ML models get decomposed signals instead of raw price — cleaner input.

5. Regime-Aware Routing (the key innovation)

HMM says: BULL regime (78% probability)
  → Increase momentum model weight
  → Decrease mean-reversion weight
  → Kelly sizing: use 0.5× (confident regime)
  → GARCH vol forecast: low → tighter targets OK

HMM says: BEAR regime (85% probability)
  → Increase mean-reversion weight
  → Decrease momentum weight
  → Kelly sizing: use 0.25× (conservative)
  → GARCH vol forecast: high → widen stop targets

HMM says: SIDEWAYS (60% probability)
  → Pairs trading signals dominate
  → Reduce overall signal count (low conviction)
  → Kelly sizing: use 0.25× (uncertain)
