CLI Interface & Project Structure

Commands, project layout, and configuration

CLI Commands
$ bist-predict --help

Usage: bist-predict [OPTIONS] COMMAND

BIST-100 Stock Market Prediction System

Commands:
  fetch      Fetch latest market data from all sources
  features   Compute features for latest data
  train      Train or retrain prediction models
  signals    Get today's trading signals
  backtest   Run walk-forward backtest on historical data
  accuracy   Show prediction accuracy history
  stocks     List tracked BIST-100 stocks
  config     View/edit configuration

Configuration (config.toml)

[data]
tcmb_api_key = ""              # Free EVDS API key
fetch_retries = 3
rate_limit_delay = 1.0         # seconds between API calls

[signals]
min_confidence = 0.70          # minimum confidence to show
lookback_days = 30             # for accuracy reporting

[models]
retrain_interval = "monthly"
ensemble_weights = "learned"   # or manual: [0.3, 0.3, 0.2, 0.2]

[backtest]
commission = 0.001             # 0.1%
slippage = 0.0005              # 0.05%

Tech Stack Summary

  • CLI: Click (Python)
  • Data: httpx + yfinance + feedparser
  • ML: XGBoost, LightGBM, PyTorch (LSTM + Transformer)
  • NLP: transformers (Turkish sentiment model)
  • Rust: PyO3 + maturin for build
  • Storage: SQLite
  • Build: uv (Python) + maturin (Rust→Python)
