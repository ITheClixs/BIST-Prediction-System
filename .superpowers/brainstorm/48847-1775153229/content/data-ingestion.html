OHLCV Price Data

Is Yatirim API (primary) Free, undocumented REST API. Most accurate BIST data. Daily OHLCV + volume. Supports all BIST-100 tickers.
Yahoo Finance (fallback) via yfinance library. BIST tickers use .IS suffix (e.g., THYAO.IS). Rate-limited but reliable backup.

Macro Data (TCMB EVDS)

USD/TRY, EUR/TRY exchange rates Daily
TCMB policy interest rate On change
CPI / Inflation data Monthly
Gold price (XAU/TRY) Daily
Government bond yields Daily

TCMB EVDS requires a free API key (register at evds2.tcmb.gov.tr)

Sentiment Data

Google News RSS Search per ticker + "borsa" keywords. Parse headlines. No API key needed.
Turkish finance news RSS Finans haber sites with RSS feeds (bloomberght, bigpara). Turkish NLP for sentiment scoring.

Data Quality Rules

  • Validate OHLCV: open/high/low/close must be logically consistent
  • Detect and handle stock splits / dividends via adjusted close
  • Flag gaps > 5 trading days (delistings, halts)
  • Rate limiter: configurable delays per source, exponential backoff on failure
  • Incremental fetch: only pull data newer than last stored date
